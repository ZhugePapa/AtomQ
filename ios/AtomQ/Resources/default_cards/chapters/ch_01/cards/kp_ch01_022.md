## 概述

智慧城市是运用信息通信技术，有效整合各类城市管理系统，实现城市各系统间信息资源共享和业务协同，推动城市管理和服务智慧化。

核心理念：==以人民为中心==，共建、共治、共享。

## 五大核心能力要素 `(23下1批)`

| 能力       | 核心含义                           |
| :------- | :----------------------------- |
| ==数据治理== | 围绕数据要素，构建数据责权利管控、开发和利用能力       |
| ==城市孪生== | 现实世界与信息世界的互动融合（数字孪生、社会孪生、设备孪生） |
| ==边际决策== | 基于决策算法和信息应用，达到快速反应、高效决策        |
| ==多元融合== | 社会关系和社会活动的动态融合，服务可编排和快速集成      |
| ==态势感知== | 对社会状态的本质反映及模拟预测，洞察可变与不可见因素     |

## 五级成熟度

| 级别 | 核心特征 |
|:---|:---|
| ==1 规划级== | 明确职责分工，初步数据采集 |
| ==2 管理级== | 明确发展战略，基础设施智能化改造，多领域单项应用 |
| ==3 协同级== | 多业务、多层级、跨领域集成，融合创新 |
| ==4 优化级== | 数据知识驱动，精准化治理，数据要素价值挖掘 |
| ==5 引领级== | 敏捷发展，三空间融合（物理+社会+信息），城市集群治理联动 |
