## 2. 全局说明

### 2.1 全局导航

- 底部 Tab 固定 3 个：`学习`、`刷题`、`我的`
- 默认进入 `学习`
- 页面标题栏默认显示当前页面标题，支持返回上一级

### 2.2 全局组件

- 顶部导航栏
- 通用骨架屏
- 通用空状态组件
- 通用异常重试组件
- 登录弹层
- 会员解锁弹层
- 底部主按钮

### 2.3 全局交互规则

- 主按钮固定使用高优先级样式，次级操作用描边按钮或文本按钮
- 关键动作需有即时反馈：加载中、成功、失败
- 用户从首页到开始学习或答题不超过 2 次点击
- 游客访问正式学习或正式练习时，统一使用登录弹层拦截

### 2.4 原型标注规范

- 页面分区按“从上到下”描述
- 交互按钮命名使用最终文案，不用“按钮 A / 按钮 B”
- 弹层、抽屉、悬浮入口单独列出

## 3. 页面流转总览

### 3.1 首次进入流转

1. 启动/首页
2. 考试日期设置页
3. 学习首页
4. 知识点卡片页或今日一练页

### 3.2 日常学习流转

1. 学习首页
2. 知识点卡片页
3. 章节练习页
4. 交卷结果页
5. 错题本页

### 3.3 主动刷题流转

1. 刷题首页
2. 章节练习页 / 今日一练页 / 真题套卷列表页
3. 答题页
4. 交卷结果页

## 4. 页面原型说明

### 4.1 启动/首页

#### 页面目标
承接首次进入用户，并让用户明确产品价值和下一步动作。

#### 页面结构

1. 顶部品牌区
内容：
- 产品名
- 简短一句价值描述

2. 中部主视觉区
内容：
- 学习计划、刷题、错题复习 3 个核心卖点
- 可选插画或简化图形

3. 底部操作区
内容：
- 主按钮：`开始学习`
- 次按钮：`先看看免费内容`

#### 关键状态

- 首次进入：展示完整引导信息
- 已有计划：跳过该页，直接进入学习首页

#### 交互说明

- 点击 `开始学习` 进入考试日期设置页
- 点击 `先看看免费内容` 进入学习首页的游客态

#### 原型备注

- 此页应轻量，不做多屏引导
- 页面高度优先保证首屏完整显示主按钮

### 4.2 考试日期设置页

#### 页面目标
完成首次进入时的唯一必要设置，用于生成学习计划。

#### 页面结构

1. 顶部说明区
内容：
- 标题：`设置你的考试日期`
- 副标题：说明系统将据此生成学习计划

2. 科目信息区
内容：
- 科目名称：`高级-信息系统项目管理师`
- 科目说明：V1 仅支持该科目

3. 日期选择区
内容：
- 日期选择器
- 默认推荐最近考试批次

4. 底部操作区
内容：
- 主按钮：`生成学习计划`
- 文本按钮：`跳过，使用默认日期`

#### 关键状态

- 默认状态：展示推荐考试日期
- 日期非法：底部提示“考试日期需晚于今天”
- 生成中：主按钮 loading

#### 交互说明

- 点击 `生成学习计划` 后调用计划生成接口
- 成功后跳转学习首页
- 失败时停留本页并提示重试

### 4.3 学习首页

#### 页面目标
让用户在 10 秒内明确今天应该学什么、做什么。

#### 页面结构

1. 顶部状态区
内容：
- 科目名
- 考试倒计时
- 可选搜索入口

2. 今日学习看板
内容：
- 今日学习任务卡
- 今日刷题任务卡
- 完成进度

3. 继续学习卡片
内容：
- 上次学习标题
- 学习位置说明
- 按钮：`继续学习`

4. 错题复习卡片
内容：
- 待复习错题数
- 按钮：`开始复习`

5. 快捷入口区
内容：
- 每日一练
- 综合知识真题
- 章节目录

#### 关键状态

- 正常态：展示任务与入口
- 无计划态：展示“开始制定计划”
- 高错题态：错题卡片上移到首屏
- 游客态：任务卡可见，但正式动作触发登录

#### 交互说明

- 点击 `继续学习` 进入知识点卡片页或未完成套题
- 点击学习任务卡进入指定知识点
- 点击刷题任务卡进入今日一练页
- 点击 `开始复习` 进入错题本页
- 点击快捷入口进入对应页面

#### 原型备注

- 首屏优先级从上到下依次为：今日任务、继续学习、错题复习、快捷入口
- 卡片不宜过多，避免首页信息过载

### 4.4 章节目录页

#### 页面目标
支持用户按章节主动浏览内容和进入学习。

#### 页面结构

1. 顶部信息区
内容：
- 章节目录标题
- 当前进度摘要

2. 章节列表区
每个章节卡片包含：
- 章节名称
- 完成进度
- 状态标签：未开始 / 学习中 / 已完成 / 已掌握
- 题量信息

3. 节列表展开区
内容：
- 节名称
- 卡片数
- 状态标签

#### 关键状态

- 加载中：章节骨架屏
- 空状态：内容准备中
- 锁定态：付费章节显示锁定标识

#### 交互说明

- 点击章卡展开/收起节列表
- 点击节项进入该节第一张知识点卡片
- 点击锁定章节触发会员解锁弹层

### 4.5 知识点卡片页

#### 页面目标
完成单个知识点学习，并承接到对应练习。

#### 页面结构

1. 顶部导航区
内容：
- 返回按钮
- 当前节标题
- 进度指示：第 N 张 / 共 N 张

2. 正文内容区
内容：
- 知识点标题
- 正文段落
- 图片
- 代码块
- 简单表格

3. 底部工具区
内容：
- 标记完成
- 去做本节练习
- 可选：收藏、笔记

4. 翻页区
内容：
- 上一张
- 下一张

#### 关键状态

- 正常态：展示完整正文
- 会员锁定态：展示摘要 + 购买引导
- 无内容态：内容待补充
- 加载失败态：重试按钮

#### 交互说明

- 点击 `标记完成` 更新当前卡片状态
- 当前节最后一张卡片完成后，弹出“去做本节练习”推荐
- 点击 `去做本节练习` 跳转章节练习页
- 左右翻页切换卡片并记录位置

### 4.6 刷题首页

#### 页面目标
支持用户主动进入各类刷题场景。

#### 页面结构

1. 顶部标题区
内容：
- 页面标题：`刷题`

2. 练习入口区
内容：
- 章节专项练习
- 每日一练
- 历年真题

3. 辅助说明区
内容：
- 今日推荐题量
- 最近练习记录入口

#### 交互说明

- 点击 `章节专项练习` 进入章节练习入口页或直接进入章节列表
- 点击 `每日一练` 进入今日一练页
- 点击 `历年真题` 进入真题套卷列表页

### 4.7 今日一练页

#### 页面目标
承接系统每日推荐题单。

#### 页面结构

1. 顶部说明区
内容：
- 今日一练标题
- 推荐逻辑说明：高频考点 / 薄弱点

2. 题单摘要区
内容：
- 题量
- 预计耗时
- 是否已完成

3. 底部操作区
内容：
- 主按钮：`开始答题`
- 次按钮：`继续上次进度`（存在未完成记录时）

#### 关键状态

- 未开始态
- 进行中态
- 已完成态：显示最近成绩与再练一次入口

#### 交互说明

- 点击 `开始答题` 进入答题页
- 若是游客且超过示例题限制，则触发登录

### 4.8 章节练习页

#### 页面目标
承接某一章节的正式练习。

#### 页面结构

1. 顶部摘要区
内容：
- 章节名
- 题量
- 预计耗时

2. 配置区
内容：
- 题量选择
- 模式说明：练习模式

3. 底部操作区
内容：
- 主按钮：`开始本章练习`

#### 交互说明

- 点击开始后进入答题页
- 当前章节无题目时展示空状态

### 4.9 真题套卷列表页

#### 页面目标
让用户按年份选择综合知识真题卷。

#### 页面结构

1. 顶部说明区
内容：
- 标题：`综合知识真题`
- 说明：按年份练习

2. 套卷列表区
每个套卷项包含：
- 年份
- 套卷名称
- 题量
- 完成状态
- 最近得分

3. 底部辅助区
内容：
- 仅供练习说明

#### 关键状态

- 无套卷：暂无可用真题
- 不完整套卷：置灰并展示“内容准备中”

#### 交互说明

- 点击套卷项进入答题页
- 若套卷未完成，优先展示“继续作答”

### 4.10 答题页

#### 页面目标
保证用户能顺畅完成练习和交卷。

#### 页面结构

1. 顶部进度区
内容：
- 返回
- 当前题号 / 总题数
- 答题卡入口

2. 题干区
内容：
- 题干正文
- 题目配图（如有）

3. 选项区
内容：
- 选项 A/B/C/D

4. 底部操作区
内容：
- 上一题
- 下一题
- 收藏
- 交卷

5. 解析区
练习模式下展示：
- 正确答案
- 文字解析

#### 关键状态

- 练习模式：答题后可即时看解析
- 套卷模式：提交前不展示解析
- 游客示例态：题数达到 3 题后触发登录
- 提交中：交卷按钮 loading

#### 交互说明

- 选择答案后立即记录本地作答状态
- 点击 `答题卡` 打开题号面板，可跳转题目
- 点击 `交卷` 后弹出二次确认
- 交卷成功跳转交卷结果页

### 4.11 交卷结果页

#### 页面目标
反馈本次练习结果，并引导下一步动作。

#### 页面结构

1. 成绩摘要区
内容：
- 分数
- 正确率
- 用时
- 错题数

2. 结果分析区
内容：
- 题目列表
- 对错标记
- 查看解析入口

3. 底部操作区
内容：
- 去看错题
- 再练一组
- 返回首页

#### 关键状态

- 正常结果态
- 结果查询中态
- 结果异常态：支持重试查询

#### 交互说明

- 点击 `查看解析` 打开题目解析详情
- 点击 `去看错题` 进入错题本页
- 点击 `再练一组` 返回上一练习入口

### 4.12 错题本页

#### 页面目标
集中管理错题，并引导复习。

#### 页面结构

1. 顶部筛选区
内容：
- 全部错题
- 高优先级
- 已掌握

2. 列表区
每个错题卡片包含：
- 题干摘要
- 最近答错时间
- 错误次数
- 当前优先级

3. 底部操作区
内容：
- 开始复习

#### 关键状态

- 无错题：展示“去开始练习”
- 未登录：展示登录引导

#### 交互说明

- 点击筛选切换列表
- 点击错题卡片进入对应题目或复习模式
- 点击 `开始复习` 进入答题页

### 4.13 我的页

#### 页面目标
集中承接个人数据与管理入口。

#### 页面结构

1. 用户信息区
内容：
- 头像昵称
- 会员状态

2. 数据看板区
内容：
- 学习天数
- 累计答题数
- 平均正确率
- 当前章节进度

3. 功能入口区
内容：
- 计划管理
- 会员中心
- 设置

#### 交互说明

- 点击 `计划管理` 进入计划管理页
- 点击 `会员中心` 进入会员页
- 点击 `设置` 进入设置页

### 4.14 计划管理页

#### 页面目标
让用户调整计划参数并重新生成后续任务。

#### 页面结构

1. 参数表单区
内容：
- 预计考试日期
- 每周学习天数
- 每日学习时长

2. 结果提示区
内容：
- 修改后对后续计划的影响说明

3. 底部操作区
内容：
- 保存并更新计划

#### 交互说明

- 点击保存后触发更新接口
- 成功后返回我的页或学习首页

### 4.15 会员页

#### 页面目标
清晰展示权益边界并承接支付。

#### 页面结构

1. 当前状态区
内容：
- 是否会员
- 有效期

2. 权益说明区
内容：
- 免费内容
- 会员内容

3. 套餐区
内容：
- 套餐卡片
- 价格
- 有效期

4. 底部操作区
内容：
- 立即开通

#### 关键状态

- 未开通态
- 已开通态
- 支付失败态
- 权益刷新失败态

#### 交互说明

- 点击套餐卡片切换选中态
- 点击 `立即开通` 拉起支付
- 支付成功后刷新权益并更新页面

### 4.16 设置页

#### 页面目标
承接系统设置和问题反馈。

#### 页面结构

1. 设置项列表
内容：
- 清理缓存
- 主题模式
- 问题反馈

2. 底部辅助区
内容：
- 版本号

#### 交互说明

- 点击 `清理缓存` 弹出确认框
- 点击 `问题反馈` 拉起反馈表单或客服入口

## 5. 关键弹层说明

### 5.1 登录弹层

#### 触发时机

- 游客点击正式学习
- 游客点击正式练习
- 游客进入错题本

#### 弹层内容

- 标题：`登录后保存学习进度`
- 说明：保存计划、错题和做题记录
- 按钮：`微信登录`、`暂不登录`

### 5.2 会员解锁弹层

#### 触发时机

- 点击会员知识点
- 点击会员题库
- 点击真题解析

#### 弹层内容

- 标题：`解锁完整内容`
- 权益摘要
- 按钮：`去开通会员`

### 5.3 交卷确认弹层

#### 触发时机

- 用户点击交卷

#### 弹层内容

- 已答题数
- 未答题数
- 按钮：`确认交卷`、`继续答题`

## 6. 线框图输出建议

### 6.1 优先绘制页面

1. 学习首页
2. 知识点卡片页
3. 答题页
4. 交卷结果页
5. 错题本页
6. 会员页

### 6.2 每张线框图至少标注

- 页面标题
- 模块名称
- 主按钮文案
- 页面状态切换
- 页面跳转去向

### 6.3 设计协作建议

- 先画低保真线框，再补视觉稿
- 优先确认主流程页面，不先做设置等低频页面
- 页面状态要与 PRD 中的异常和权限规则对齐
