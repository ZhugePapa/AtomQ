## 概述

软件的生命周期通常包括以下 ==7 个阶段==：

<!-- render:native caption:软件生命周期各阶段 -->
| 阶段 | 主要活动 |
|:---|:---|
| 1. ==可行性分析与项目开发计划== | 确认项目是否值得做，制定开发计划 |
| 2. ==需求分析== | 明确用户需求，编写需求规格说明书 |
| 3. ==概要设计== | 系统架构设计，模块划分 |
| 4. ==详细设计== | 模块内部详细设计 |
| 5. ==编码== | 将设计转换为程序代码 |
| 6. ==测试== | 发现和修复缺陷 |
| 7. ==维护== | 交付后的修改和完善 |
