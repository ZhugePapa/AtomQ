### 概述

车联网（IoV）是新一代网络通信技术与汽车、电子、道路交通运输等领域深度融合的新兴产业形态。

智能网联汽车：搭载先进传感器、控制器、执行器 + 通信与网络技术 → 实现车与车、路、人、云端的信息交换共享。

### 端-管-云三层体系

| 层级 | 功能 | 说明 |
|:---|:---|:---|
| ==端系统== | 智能传感器、通信终端 | 感知行车状态、具备IoV寻址和可信标识 |
| ==管系统== | 异构网络通信与漫游 | 解决V2V、V2I、V2N、V2P互联互通 |
| ==云系统== | 车辆运行信息平台 | 物流、客货运、保险、4S店等多源信息汇聚 |

### 五种链接方式

| 方式 | 说明 |
|:---|:---|
| ==车与云== | 卫星/蜂窝通信 → 平台 → 数据共享 |
| ==车与车(V2V)== | 车辆间信息交互（位置、速度） |
| ==车与路(V2I)== | 道路固定设施通信 → 路径引导 |
| ==车与人(V2P)== | Wi-Fi/蓝牙/蜂窝 → 用户监测控制 |
| ==车内设备== | 设备间通信 → 数字化控制系统 |
