### 概述

广域网协议包括 ==PPP==、==ISDN==、==xDSL==、==DDN==、==X.25==、==FR 帧中继==、==ATM 异步传输模式==。

IEEE 802 协议中，==802.3== 对应以太网的 CSMA/CD 载波监听多路访问/冲突检测协议，==802.11== 对应无线局域网 WLAN 标准协议。

### 高频对照

| 协议/标准 | 记忆点 |
|:---|:---|
| PPP | 点对点协议 |
| ISDN | 综合业务数字网 |
| xDSL | 数字用户线路统称 |
| DDN | 数字专线 |
| FR | 帧中继 |
| ATM | 异步传输模式 |
| 802.3 | 以太网、CSMA/CD |
| 802.11 | WLAN、无线局域网 |

### 易混点

802.3 和 802.11 是 IEEE 局域网/城域网标准族中的常考项，不要和 WAN 协议列表混为一类。
