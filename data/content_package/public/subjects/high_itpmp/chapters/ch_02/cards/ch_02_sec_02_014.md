### 概述

NoSQL 常见类型包括 ==键值数据库==、==列存储数据库==、==面向文档数据库==、==图形数据库==。

### 核心类型

| 类型 | 学习抓手 |
|:---|:---|
| 键值数据库 | 以 key-value 方式存储，适合缓存、会话等快速读写场景 |
| 列存储数据库 | 面向列或列族存储，适合大规模分布式数据 |
| 面向文档数据库 | 以 JSON / BSON 等文档组织数据，结构灵活 |
| 图形数据库 | 用节点和边表达关系，适合社交关系、知识图谱等场景 |

### 易混点

“列存储数据库”和“关系型数据库中的列”不是同一考点；题目通常考 NoSQL 类型枚举。
