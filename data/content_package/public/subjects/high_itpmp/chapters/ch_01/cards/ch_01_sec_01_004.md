#### 概述

信息质量属性主要包括 ==精确性、完整性、可靠性、及时性、经济性、可验证性和安全性==。

<!-- render:native caption:信息质量属性 -->
| 属性 | 判断标准 |
|:---|:---|
| ==精确性== | 是否准确反映事实 |
| ==完整性== | 是否包含必要内容 |
| ==可靠性== | 来源和内容是否可信 |
| ==及时性== | 是否在需要时可获得 |
| ==经济性== | 获取和维护成本是否可接受 |
| ==可验证性== | 能否被证实或证伪 |
| ==安全性== | 是否防止泄露、篡改和未授权访问 |

#### 场景考点

对==金融信息==而言，最重要的特性通常是==安全性==；对==经济与社会信息==而言，最重要的特性通常是==及时性==。

#### 易混点

“可靠性”强调可信赖，“安全性”强调保密、完整与授权控制。金融信息优先选安全性，不要被“可靠性”干扰。
