### 概述

信息的特征主要包括 ==客观性==、==普遍性==、==无限性==、==动态性==、==相对性==、==依附性==、==变换性==、==传递性==、==层次性==、==系统性== 和 ==转化性==。

<!-- render:scroll caption:信息的 11 个特征 -->
| 特征 | 记忆解释 |
|:---|:---|
| 客观性 | 信息反映客观事物，不以人的意志为转移 |
| 普遍性 | 自然界和人类社会中普遍存在信息 |
| 无限性 | 信息可不断产生和扩展 |
| 动态性 | 信息会随时间和环境变化 |
| 相对性 | 同一信息对不同主体价值不同 |
| 依附性 | 信息必须依附于载体 |
| 变换性 | 信息可以加工、转换和重组 |
| 传递性 | 信息可以跨时间、空间传递 |
| 层次性 | 信息可按战略、管理、操作等层次组织 |
| 系统性 | 信息之间相互联系，形成整体 |
| 转化性 | 信息可转化为知识、决策和生产力 |

### 高频理解

软考不常让你解释每一个特征，但会把场景对应到特征。例如“同一报告对项目经理和普通用户价值不同”对应相对性；“信息必须记录在纸张或系统中”对应依附性。
